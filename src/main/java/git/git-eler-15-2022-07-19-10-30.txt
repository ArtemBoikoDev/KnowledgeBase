+15 - Git reset (откат закоммиченного)
{t 5-39} [in-KB git-eler-15-2022-07-19-10-30]








00-00...01-23

git add
чтобы перевести файл в индекс

git commit
чтобы из индекса сделать коммит

git reset --soft
перевести из закомиченного обратно в индекс

если мы не прописываем --soft, по умолчанию используется --mixed стратегия

git reset --mixed
закоммиченное минуя индекс возвращается в файловую систему








01-23...до конца

git reset --hard
удаляет все изменения

git reset --soft HEAD~1
файл возвращён в предыдущее состояние и возвращён в индекс

git reset --mixed HEAD~1
состояние файла вернулось и он не теперь добавлен в индекс

git reset --hard HEAD~1
просто удаляет коммиты
